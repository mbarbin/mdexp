Tutorials are hands-on, learning-oriented guides that walk you through a
complete project from start to finish. Each tutorial builds something
concrete while teaching mdexp concepts along the way.

## [The Potion Brewery](potion-brewery/)

A config-as-code tutorial where both documentation and JSON configuration
files are generated from the same shared OCaml values. Demonstrates literate
programming with mdexp, table rendering, and build-verified config generation.

**Authors:** Mathieu Barbin, Claude Opus 4.6
